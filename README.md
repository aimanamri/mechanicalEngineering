## ▶️ Getting Started

Starting with **git clone**:

```bash
git clone https://github.com/aimanamri/mechanicalEngineering.git -b main
```

## Pre-Requisites
- Python 3
- Jupyter Notebook
- Install all packages in requirements.txt
```bash
pip install -r requirements.txt
```
